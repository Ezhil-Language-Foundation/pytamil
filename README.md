# Why Pytamil
Core philosophy of the library is to clearly separarte tamil language conepts from the programming language. For example, Tamil рокрпБрогро░рпНроЪрпНроЪро┐ rules are captured in human readable text file [рокрпБрогро░рпНроЪрпНроЪро┐ро╡ро┐родро┐роХро│рпН.yaml](pytamil/родрооро┐ро┤рпН/рокрпБрогро░рпНроЪрпНроЪро┐ро╡ро┐родро┐роХро│рпН.yaml ) in YAML format. This allows people with no prior knowledge in computer programming to contribute to the project and have more meaningful and natural discussion on the language concepts.
